URL Shortening Service

Usage:

Create shorter URLs! Well, they're still kinda long because I'm cheap and won't buy a domain.

To create a URL shortcut:

Follow this service's URL with /new and the URL that you would like to shorten. You can include the http:// or leave it out.

  • For example: https://desolate-inlet-99642.herokuapp.com/new/www.iamthecoolestwebsiteever.com/scooooobiedoooobiedooooooo

This will return a shortened URL for you to copy and bookmark.

To access your shortcut:

Simply go to the shortened URL that you received and the service will redirect you to your website
